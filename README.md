# Responsive Landing Page

A responsive landing page created as part of my Web Development Internship at Prodigy InfoTech. The project focuses on building a clean, user-friendly webpage with an interactive navigation menu and a layout that works across different screen sizes.

## Features

- Responsive webpage layout
- Interactive navigation menu
- Smooth and simple user experience
- Mobile-friendly design
- Clean and organized interface

## Technologies Used

- HTML
- CSS
- JavaScript

## Purpose

This project was created to strengthen my frontend development skills and gain practical experience in building responsive and interactive web pages.
