About EduSpark

Designed for students who want to start simple and grow steadily.

EduSpark is a responsive landing page project created to present a modern online learning platform. The page focuses on clean structure, smooth navigation, interactive styling, and mobile-friendly design.

The goal of this project is to demonstrate how HTML, CSS, and JavaScript can work together to create a professional and user-friendly landing page.
